# فروشگاه بزرگ شیراز

این ربات به صورت خودکار توسط ربات‌ساز ساخته شده است.

- نوع ربات: فروشگاهی
- امکانات: پرداخت کارت‌به‌کارت و تایید رسید، مدیریت محصول/خدمت، چند ادمین، پنل مدیریت ساده، درگاه پرداخت آنلاین، گزارش‌گیری
- پشتیبانی: ارتباطی

## نکته
این نسخه، MVP عملیاتی است: فرم، ثبت سفارش، رزرو، تیکت و ارسال پیام به ادمین را انجام می‌دهد. اتصال درگاه پرداخت آنلاین واقعی نیاز به اطلاعات پذیرنده و API درگاه دارد.
